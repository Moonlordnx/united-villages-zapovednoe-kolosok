## 📌 О проекте

Этот репозиторий содержит код **официального веб-сайта объединённого представительства** двух дачных посёлков:

- **🌲 ДПК «Заповедное»** – тихий лесной массив, экотропы, заповедная атмосфера  
- **🌾 ДПК «Колосок»** – открытые поля, просторные участки, соседское общение  

> 🎯 Главная цель сайта — создать единое цифровое пространство для жителей, правления и администрации.

## 🚀 Быстрый старт

### 1️⃣ Клонировать репозиторий

```bash
git clone https://github.com/your-username/zapovednoe-kolosok-union.git
cd zapovednoe-kolosok-union
```

### 2️⃣ Установить зависимости (если нужны)

```bash
npm install
# или
yarn install
```

### 3️⃣ Запустить локально

```bash
npm start
# или откройте index.html
```

Сайт будет доступен по адресу:  
➡️ `http://localhost:3000`
